## INSTALLATION

1. CLONE THE REPOSITORY
   ```
   git clone https://github.com/yourusername/brutagram.git
   ```

2. INSTALL DEPENDENCIES
   ```
   cd brutagram
   npm install
   ```

3. SET UP ENVIRONMENT VARIABLES
   ```
   cp .env.example .env
   ```
   EDIT `.env` WITH YOUR CONFIGURATION

4. RUN THE APPLICATION
   ```
   npm start
   ```
